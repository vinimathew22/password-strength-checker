## Password Strength Checker
A simple, user-friendly Password Strength Checker application built in Python using tkinter for the graphical user interface.

# Features

Real-time password strength evaluation as you type.

Checks for:
Minimum length of 8 characters
Presence of lowercase letters
Presence of uppercase letters
Presence of digits
Presence of special characters

Strength levels:
Very Weak (red)
Weak (red)
Medium (orange)
Strong (green)
Very Strong (dark green)
